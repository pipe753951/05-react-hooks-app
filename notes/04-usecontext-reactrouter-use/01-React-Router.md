# #01. ↗️ React Router

React Router es un paquete que proporciona rutas para React. Este paquete implementa tres estrategias para trabajarlo: El modo declarativo (_declarative_), el modo de información (_data_), y el modo entorno de trabajo (_framework_):

1. El modo declarativo provee funciones de ruta básicas.
2. El modo de información mueve la configuración de rutas fuera del _renderizado_ de React, permitiendo más características.
3. El modo entorno de trabajo pasa el de información a un plugin de Vite para añadir toda la funcionalidad de estos sistemas.

El instructor recomienda el último para aplicaciones ligeras del lado del servidor; y el resto para SPAs. El modo declarativo se usa de la misma manera de la que se acostumbra a trabajar en React (O, por lo menos, a nivel de la sección); y, el modo de información ofrece, como se mencionó anteriormente, más características.

El instructor utilizará el modo de información, el resto los parta para un curso de React Router.
